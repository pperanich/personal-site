This is my first blog post! I built this site using the **AHA Stack**: Astro, htmx, and Alpine.js.

## Why AHA?

The AHA Stack embraces a return to simplicity:

- **Astro** handles server-side rendering and file-based routing
- **htmx** enables dynamic server interactions without writing JavaScript
- **Alpine.js** adds lightweight client-side interactivity where needed

It's HTML-first, fast by default, and a joy to work with.
